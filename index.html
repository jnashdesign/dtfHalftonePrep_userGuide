Requirements

Adobe Photoshop 23.3+ macOS or Windows Creative Cloud Desktop

Installation

Easiest method (all versions)

  1. Make sure the Creative Cloud Desktop app is running.
  2. Double-click the .ccx file you received.
  3. Creative Cloud will open and show a confirmation dialog. Click Install to continue.
  4. Open (or restart) Photoshop.
  5. Go to Plugins > Print Prep Studio > Print Prep Studio to open the panel.
The plugin appears under Plugins > Print Prep Studio in the Photoshop menu bar.
Note: When installing a .ccx file, Creative Cloud may show a warning that the plugin doesn't come from the Creative Cloud Marketplace. This is normal — click Install to proceed.

If double-clicking doesn't work

macOS
  1. Right-click the .ccx file and choose Open With > UnifiedPluginInstallerAgent.
  2. If you see a macOS security warning, go to System Settings > Privacy & Security and click Open Anyway.
Windows
  1. Make sure the .ccx file is on the same drive as Photoshop (e.g. your Desktop or Downloads folder).
  2. Try double-clicking again. If it still doesn't work:
    1. Use WinRAR or 7-Zip to extract the .ccx file (it's a zip archive).
    2. Copy the extracted folder into your Photoshop install directory:
      C:\Program Files\Adobe\Adobe Photoshop [Version]\Plug-ins\
    3. Restart Photoshop.

After installation

The plugin panel will appear on the right side of your workspace. You can dock it, float it, or resize it however you like.

Still not showing up? Try signing out of Creative Cloud, restarting your computer, signing back in, then reopening Photoshop.
Tip: Dock the panel where you want it, then go to Window > Workspace > New Workspace to save your layout. Photoshop will remember the position for future sessions.

Source Layer

All tabs (Halftone, Vinyl Prep, and Screen Print) share the same Source Layer selector at the top of the plugin. Choose which layer in your document the effect should be applied to. If your file has multiple layers, use the dropdown to select the one with your artwork. Click the Refresh button if you've added or renamed layers since opening the plugin.

Locked layers: If the source layer is locked (like the default "Background" layer), the plugin will automatically unlock it for you and let you know with a notification.
Step-by-step workflow: Some actions stay disabled until the required previous step is completed. Hover a disabled button to see exactly what needs to be done first.

Export Size (Shared)

The Export Size section is global and applies to all export actions across the plugin. Set it once, then export from any tab.

  • Size mode — Choose Inches + DPI or Pixels.
  • Width/Height/DPI — Control final export dimensions used by Halftone PNG and Screen Print separation exports.
  • The fields auto-sync to your active document size by default, and can be edited manually at any time.

Halftone

The Halftone tab creates dot or line halftone patterns for DTF (Direct to Film) printing.

Quick Start

  1. Open an image in Photoshop.
  2. In the plugin, pick the Source Layer you want to apply the halftone to.
  3. Set your Base Color to match the garment or substrate you're printing on.
  4. Adjust the Halftone Settings and Levels to taste.
  5. Click Create Halftone to preview the result.
  6. When you're happy, click Apply Halftone as Mask to apply it to your original image.
  7. Set output dimensions in the shared Export Size section.
  8. Use the Halftone Export for DTF Film button to save a print-ready transparent PNG.

Base Color

Set the color of the shirt or substrate you're printing on. This creates a "Base Color" layer behind your artwork so you can preview how the halftone will look on the final product. The default is black (#000000).

  • Hex input — Type a hex color code (e.g. #000000 for black, #FFFFFF for white) and click Set Base Color.
  • Eyedropper (dropper icon) — Click the eyedropper button, then click anywhere on the canvas to sample a color.
  • Color swatch (colored square) — Click the square to open a color picker where you can visually choose a color from the spectrum.
  • Show shirt color preview — Toggle this on/off to show or hide the base color layer behind your artwork.

Halftone Settings

Setting What it does
Type Dot creates circular halftone dots. Line creates parallel line patterns.
Dot / Line Size Controls how large the dots or lines are. Larger values = bigger pattern elements.
Angle Rotates the halftone pattern (0° to 90°).
Threshold Controls the cutoff between black and white. Lower values = more white. Higher values = more black. Most useful for the Line type.
Invert Halftone Swaps the black and white in the halftone pattern. Use this when printing on dark shirts — the dots represent where ink IS applied instead of where it ISN'T.

Levels

Fine-tune the tonal range of your image before the halftone is applied. These work just like Photoshop's Image > Adjustments > Levels dialog.

Slider What it does
Shadows Sets the black point (0–253). Slide right to crush darker tones to pure black. Default: 0.
Midtones Adjusts the gamma / midpoint. Slide left to darken the midtones, right to brighten them. Default: 1.00 (no change).
Highlights Sets the white point (2–255). Slide left to push brighter tones to pure white. Default: 255.
Tip: Try pulling the Shadows up and Highlights down to increase contrast before halftoning — this often produces cleaner, more defined patterns.

Preview & Mask

  • Live Preview — When enabled, the halftone updates automatically as you change settings. Turn this off if your computer feels slow while adjusting sliders.
  • Create Halftone — Generates a halftone preview layer. After the first click, the button changes to Update Halftone so you can re-apply after tweaking settings.
  • Apply Halftone as Mask — Applies the halftone pattern as a layer mask on your original image. This is your "commit" step — the halftone controls which parts of your artwork are visible. The preview layer is hidden automatically.
  • Reset to Defaults — Resets all sliders and settings back to their defaults and removes any halftone preview layers.

Export

Save your finished artwork as a transparent PNG ready for DTF printing.

  • Click Export for DTF Film to save the file. You'll be prompted to choose where to save it.
  • The exported size comes from the shared Export Size section (outside the tabs).
  • If export is disabled, hover the button to see the required prerequisite.

Halftone Tips

Dark shirts

Check "Invert Halftone" so the dots represent where ink is printed. White ink goes down first, then the halftone pattern is printed on top.

Light shirts

Leave "Invert Halftone" off. The dots represent transparent areas where the shirt color shows through.

Dot size matters

For high-resolution images (above 2000px), the default dot size is 24. For smaller images, it's 8. Larger dots are bolder, smaller dots are more detailed.

Line angle

A 45° angle is the standard for halftones. 0° gives horizontal lines, 90° gives vertical lines.

No half-pixels

The plugin automatically ensures every pixel is either fully opaque or fully transparent — no soft edges that could cause printing issues.

Use Levels for contrast

Instead of a simple contrast slider, use the Levels section to fine-tune shadows, midtones, and highlights — just like you would in Photoshop's Levels dialog.

Vinyl Prep

The Vinyl tab reduces your image to a small number of flat colors and creates cutter-ready vector shapes, ready to export as SVG for vinyl cutting machines like Cricut, Silhouette, and others.

Quick Start

  1. Open an image in Photoshop.
  2. Pick the Source Layer with your artwork.
  3. Adjust the Vinyl Settings (colors, smoothing, detail, tolerance).
  4. Click Create Vinyl Shapes to process the artwork and build shape layers.
  5. Adjust any slider and click Create Vinyl Shapes again to iterate until you're happy.
  6. Click Export SVG to save a cutter-ready SVG file with all colors.

Vinyl Settings

Setting Range What it does
Max Colors 2 – 5 The maximum number of flat colors in the final result. The plugin analyzes your image, reduces it to this many colors, and separates each color into its own layer.
Edge Smooth 0 – 5 Applies a median filter to clean up noise and soften jagged edges while keeping hard color boundaries crisp. Higher values = smoother edges, but may lose fine detail.
Min Detail 0 – 20 Removes features that are too small for a vinyl cutter to cut or weed. Higher values = more aggressive filtering. Set to 0 to keep all detail.
Tolerance 1 – 20 Controls how closely the vector paths follow the pixel edges. Lower values = more anchor points, more accurate. Higher values = fewer points, smoother curves that are easier for cutters to follow.

Create Vinyl Shapes

Click Create Vinyl Shapes to run the full pipeline:

  1. Color reduction — The image is posterized and reduced to the specified number of colors.
  2. Shape creation — Each color is converted into a solid-color vector shape layer with cutter-friendly edges. Small details are filtered out based on Min Detail.
  3. Final layer structure — The workflow keeps only shape layers inside a single "Vinyl Shapes" folder (pixel prep layers are removed).

The shape layers are immediately visible in Photoshop so you can inspect the result. Adjust any slider and click Create Vinyl Shapes again — the previous result is automatically replaced.

After previewing, color chips appear below the button showing which colors were detected.

If the image has fewer significant colors than requested, the plugin uses the actual detected color count and reports it in the status message (instead of creating random extra color layers).

Export

  • Export SVG — Saves all color shapes as a single SVG file with fill colors and proper alignment. Compatible with Adobe Illustrator, Cricut Design Space, Silhouette Studio, Inkscape, and any software that opens SVG files.
  • Reset Vinyl Prep — Removes all vinyl prep layers (color separations, shape layers, and the working layer) and resets the plugin to a clean state.
  • Export dimensions come from the shared Export Size section.

Vinyl Tips

Start with fewer colors

Try Max Colors at 2 or 3 first. You can always increase if the result is too simplified.

Adjust Min Detail for your cutter

If your cutter struggles with small shapes, increase Min Detail to filter them out. If you're losing important detail, lower it.

Tolerance is your friend

A tolerance of 3–5 works well for most vinyl cutting. Go higher (8–12) for very smooth shapes. Go lower (1–2) if accuracy matters more.

Edge Smooth cleans up noise

Even 1px of smoothing removes stray single-pixel artifacts. Increase to 3–5 for noisy or low-resolution source images.

Check shapes before exporting

After clicking Create Vinyl Shapes, zoom in on the shape layers. You can manually edit individual shapes in Photoshop before exporting.

SVG preserves colors

The exported SVG includes the fill color for each shape, so your cutter software knows which vinyl color to use for each layer.

Screen Print

The Screen Print tab creates spot-color halftone separations with per-channel angles and optional white underbase, then exports each screen as a black-on-transparent PNG.

Quick Start

  1. Open an image in Photoshop.
  2. Pick the Source Layer with your artwork.
  3. Adjust Max Colors, Edge Smooth, Dot Size, and Underbase.
  4. Click Create Screens to generate separation layers.
  5. Inspect the colored separation preview layers inside the Screens folder.
  6. Set output dimensions in shared Export Size, then click Export Screens.

Screen Print Settings

Setting What it does
Max Colors Maximum number of spot colors to keep for separation.
Edge Smooth Applies median smoothing before separation to clean noise and jagged transitions.
Dot Size Controls effective halftone dot scale for all separated colors.
Underbase Adds a white underbase screen where non-black inks print for better opacity on dark garments.

Layer Structure

  • Each detected spot color becomes a solid color fill layer with a halftone mask.
  • Separations are grouped in a folder named "Screens".
  • The optional underbase is a white fill layer named "Screen Underbase".
  • The prep/working layer is named "Screen Prep" and hidden automatically.

Export Screens

  • Export Screens writes one PNG per separation (including underbase when enabled).
  • Each file is black ink on transparent background, ready for film output / burning screens.
  • Export dimensions come from the shared Export Size section.
  • If the export button is disabled, hover it for the prerequisite message.

FAQs

General

The plugin says it unlocked my layer — is that OK?

Yes. If your source layer is locked (like the default "Background" layer), the plugin will automatically unlock it so it can work. You'll see a notification letting you know. This is safe and you can re-lock the layer afterward if needed.

Why is a button disabled?

Some actions require a previous step to be completed first. For example, Apply Halftone as Mask requires Create Halftone, and export actions require their preview/build step first. Hover the disabled button to see what action is required.

Why is the layer list empty or outdated?

Click the Refresh button in the Source Layer section. The list also updates automatically when you add or remove layers, but a manual refresh can help.

Why is the plugin panel cut off?

The panel is scrollable — scroll down to see all sections. You can also resize the panel by dragging its edges.

Halftone

Why does the halftone preview look wrong after editing?

Click Update Halftone to regenerate with your current settings. The preview doesn't auto-update when you edit the source image outside the plugin.

Why does the eyedropper button time out?

You have 15 seconds to click a color on the canvas after pressing the eyedropper button. Try again if it times out.

Can I undo the halftone mask?

Yes. Use Edit > Undo (Cmd/Ctrl+Z) in Photoshop immediately after applying. You can also click Reset to Defaults in the plugin to remove all halftone layers.

What are the Levels sliders for?

They work just like Photoshop's Levels adjustment (Image > Adjustments > Levels). Shadows controls the black point, Midtones adjusts the gamma, and Highlights controls the white point. Use them to fine-tune the tonal range before the halftone is applied for cleaner results.

Vinyl Prep

The vinyl preview has too few or too many colors

Adjust the Max Colors slider and click Create Vinyl Shapes again. The plugin will re-analyze the image and reduce to the new color count.

The vinyl shapes are too small or too detailed for my cutter

Increase Min Detail to filter out features that are too small to cut or weed. Increase Tolerance for simpler curves with fewer anchor points.

The Export SVG button gives an error

Make sure you've clicked Create Vinyl Shapes first to generate the shape layers. The export reads data from those shape layers.

Can I edit the shapes before exporting?

Yes. After clicking Create Vinyl Shapes, the shape layers appear in your Photoshop layers panel. You can use Photoshop's path tools to adjust individual shapes before exporting.

What software can open the exported SVG?

The SVG file works with Adobe Illustrator, Cricut Design Space, Silhouette Studio, Inkscape, and any other application that supports SVG files. Each color is a separate path element with its fill color preserved.

Screen Print

Why is Export Screens disabled?

Click Create Screens first. Export reads the generated separation layers and outputs one PNG per screen.

Are screen exports colored?

No. Canvas preview layers are colored for readability, but exported files are black-on-transparent for film output.

Where do output dimensions come from?

Dimensions are controlled by the shared Export Size section and apply to both Halftone PNG export and Screen Print exports.
